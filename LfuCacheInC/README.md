# LfuCache:  Least Frequently Used Cache

Doubly linked list + Hashtable

A C++ version can be found in [LfuCacheInCpp](../LfuCacheInCpp/README.md).

### How to use

```sh
LfuCache$ make CFLAGS="-fsanitize=address -O3 -Wall -g"

LfuCache$ ./main

```
